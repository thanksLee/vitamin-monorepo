## 프론트엔드(FE) 태스크

| 번호 | 기능 | 상세 내용 | 우선순위 |
| :-- | :-- | :-- | :-- |
| FE-1 | 로그인 화면 구현 | - 아이디/비밀번호 입력 필드<br>- 로그인 버튼<br>- 테마 변경 버튼(Light/Dark)<br>- 언어 선택 버튼(kr/en/zh) | 높음 |
| FE-2 | 입력 유효성 검증 | - 아이디 필수 입력 검증<br>- 비밀번호 형식 검증(6-30자, 영문자+숫자)<br>- 오류 메시지 표시 로직 | 높음 |
| FE-3 | 로그인 요청 처리 | - 사용자 입력 데이터 수집<br>- API 호출 구현<br>- 응답 처리 로직 | 높음 |
| FE-4 | 토큰 관리 | - Access/Refresh 토큰 Session Storage 저장<br>- 토큰 만료 감지<br>- 토큰 갱신 요청 처리 | 높음 |
| FE-5 | 테마 관리 | - 테마 변경 기능 구현<br>- 테마 상태 저장<br>- 테마별 스타일 적용 | 중간 |
| FE-6 | 다국어 지원 | - 언어 변경 기능 구현<br>- 다국어 리소스 관리<br>- 언어별 텍스트 적용 | 중간 |
| FE-7 | 오류 처리 | - 로그인 실패 알림 표시<br>- 계정 잠금 알림 표시<br>- IP 차단 페이지 구현 | 높음 |
| FE-8 | WebSocket 연결 | - 멀티 로그인 감지 리스너 구현<br>- 자동 로그아웃 처리<br>- 모달 표시 및 확인 버튼 처리 | 중간 |
| FE-9 | 에러 핸들링 프레임워크 | - 중앙화된 에러 처리 로직 구현<br>- 에러 코드별 UI 처리 분리<br>- 에러 메시지 표시 컴포넌트 | 중간 |
